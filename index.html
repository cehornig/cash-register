JavaScript Cash Register

Based on the JavaScript cash register project from freeCodeCamp Instructions:

Enter the price of the item, the contents of your cash register, and the change given by the customer below. All values should be in dollars; for example, if you have 50 pennies, enter 0.5 in the PENNY category of the cash register input. If you enter an impossible value--for example, $10 worth of $20 bills--the cash register function will return an error. Otherwise, it will return the change due in the correct denominations, and whether the cash register has been emptied. If it can't return enough change, it will return "INSUFFICIENT FUNDS" instead.
